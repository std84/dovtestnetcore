this is project version with .net core there is another with graphQL   aproch
the project is done in visual code enviormand so need to run in two parts
how to run the project:
1) download the project from git hub
2) install the node modules of the angular part
3) go to dovtest-spa folder in the command line and run the angular side of the project with ng serve
4) go to dovtest-api folder in another command line and run it with dotnet run
